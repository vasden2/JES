Movie Objects in JES

To understand how to work with movies in JES, you must first understand the objects (or encodings) that represent movies.

You can imagine that each movie is made up of a set of frames.
Movies
 Movies are a set of frames, either a list of pictures, or stored in formats such as MOV and AVI.
Frames

A frame is a single image in a movie. Each frame in a movie is slightly different than the ones next to it.
