Installing JES on Mac OS X

These instructions will get JES installed on your Mac OS X computer. Unfortunatley, video playback will not work within JES on Macintosh Platforms. All Mac systems come with QuickTime, though. Almost every video format can be played back through QuickTime.

1. You should have downloaded JES for the Mac as a disk-image file (.dmg). Double-click that file to open it up, and find the JES application package. Drag the JES application package to the Applications folder on your hard drive.

2. Double-click the JES application in your Applications folder to start up JES! You may want to drag JES on to your Dock as well.

3. You can now safely "eject" the JES disk image and delete the .dmg file, but you don't have to.
