Troubleshooting
These are a few of the most common problems encountered by MediaTools users:

Q. When I try to close the MediaTools window, it asks: "Quit Squeak without saving?" even though I already saved what I was working on.
This message is caused by the fact that Squeak is not just a program--it's a programming environment too. Since you are not worried about programming in Squeak, you don't need to worry about saving changes in Squeak. Go ahead and select "yes" and the program will quit.
 
Q. I accidentally dragged an object from one of the tools into another. Now I can't move it back without making copies of it!
Again, because Squeak is a programming environment, it is fairly easy to change the way it looks. Just quit MediaTools and start it up again to find everything back where it belngs.
