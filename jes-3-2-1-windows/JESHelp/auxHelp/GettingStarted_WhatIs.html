What is JES?

The Jython Environment for Students, or JES, is an easy-to-use programming environment that allows you to open and edit existing Jython programs or create your own.

JES was developed to help you write Jython programs that manipulate media, so it understands lots of commands related to sound, images, and video. The names for these commands and information about using them in your programs can be found in the help section for each type of media.
